# 🚦 Real-Time Traffic Prediction System (AI + Data Science)

## 📌 Overview

This project is an AI-powered Real-Time Traffic Prediction System that predicts traffic congestion levels and suggests the best route using Machine Learning and Data Science techniques.

The system simulates real-world traffic conditions using historical and dynamic data to help users avoid congestion and reduce travel time.

---

## 🎯 Key Highlights

* Predicts traffic congestion (Low / Medium / High)
* Suggests best route using shortest-path algorithms
* Provides Estimated Time of Arrival (ETA)
* Interactive dashboard using Streamlit
---

## ✨ Features

* 🚦 Traffic Prediction based on time, location, and vehicle count
* 🗺️ Route Optimization using Dijkstra / A* algorithm
* ⏱️ ETA Calculation
* 📊 Data Visualization (EDA graphs, peak hours)
* ⚡ Real-Time Simulation of traffic
* 🔍 Feature Importance (Explainable AI)

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn, XGBoost
* Matplotlib, Seaborn, Plotly
* Streamlit

---

## 📊 Dataset

* Synthetic + real-world inspired traffic dataset
* Features used:

  * Timestamp (hour, day)
  * Location
  * Vehicle count
  * Traffic speed
  * Weather conditions

## 🤖 Machine Learning Models

📈 Evaluation Metrics:

* RMSE
* MAE
* R² Score
